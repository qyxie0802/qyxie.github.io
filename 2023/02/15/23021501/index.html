OpenAI注册，ChatGPT使用指南

发表于 2023-02-15 | 分类于 实用教程 | | 本页总阅读数 次
字数统计: 1.2k | 阅读时长 ≈ 4

ChatGPT是由OpenAI开发的一个人工智能聊天机器人程序，ChatGPT能够通过学习和理解人类的语言来进行对话，还能根据聊天的上下文进行互动，真正像人类一样来聊天交流，甚至能完成撰写邮件、视频脚本、文案、翻译、代码等任务。

但是ChatGPT在国内使用起来是非常不方便，需要外国的网络环境和手机号。

使用ChatGPT有两种方法：

网页在线模拟：

您可以在线模拟ChatGPT，并通过在网页上与其进行交互来使用它。网页在线使用的是GPT3.5 （davinci-003），心智理论相当于相当于9岁儿童！

注册 OpenAI

Tips:这步需要科学上网，要求是韩国，日本，印度，新加坡，美国这些国家的地址都合适。反正香港或中国的代理肯定不行。不然会提示“OpenAI’s services are not available in your country. “

首先打开ChatGPT的账户注册页面，建议配合谷歌或者微软食用。再不济邮箱注册。

注册中需要验证手机号码，同样提交中国号码会出现说不能在当前国家服务的提示。需要国外手机号。可以借助接码平台。

接码平台

打开接码平台 sms-activate.org，使用邮箱注册一个账号

这步不需要科学上网，遇到打不开问题请更换浏览器，更换设备，更换网络环境解决。

遇到注册确认邮件问题请看邮箱垃圾箱或者耐心等待，邮件可能有延迟（我就是等了大概十分钟才收到注册确认邮件）。

登录，右上角头像充值

一次接码OpenAi的验证码费用好像是15卢布，冲1美金，约等于7人民币，70卢布，也可以需要多少冲多少。选支付宝支付。充值到账有一点点延迟。

选择服务->OpenAI

就买第一个印尼的，亲测可用。小黄车点一下就好，不要多点

然后激活->复制系统分配的手机号填入刚OpenAI注册界面，点击【Send code via SMS】发送验证码按钮

tips:我们买的是一次性号码，只有20分钟有效期，请及时操作。

在接码平台收到验证码，回填完成注册。

然后访问chat.openai.com/chat，通过网页与其进行交互来使用它

通过API：

您也可以通过访问OpenAI的API来使用ChatGPT。您只需要向API发送一个询问，然后您将得到一个回答。您可以使用各种编程语言（例如Python，Java，Ruby，C ++等）来实现这一点。但OpenAI的API使用的是davinci-002版本的GPT3，跟GPT3.5相比，只能说差强人意。不建议不推荐不好玩。

简单举例使用在VsCode中使用ChatGPT

1.下载安装打开 Visual Studio Code

2.单击左侧栏中的扩展程序图标

3.搜索安装“Chinese”中文包，”ChatGPT中文版”

4.点击ChatGPT扩展对话框输入即可对其进行提问。

模式切换：目前插件支持两个模式

国内模式：此模式安装插件后自动开启，免费、无需注册、登录、代理，开箱即用。但是不知道调用的啥，感觉有猫饼，我问它“10900K睿频多少”，它说“9.9GHz”。。。。。。我直接全人类谢谢你。。。。。。（但是人家免费！！！！！！）

国外模式:此模式需要手动切换，调用OpenAI的API接口，有以下特点。

需要注册 OpenAI 的账号获取key。

此模式下，速度会比较稳定，如果有条件，建议使用本模式！！！！！！

输入key登录成功后，不再需要开启代理。

具体操作如下：

科学上网注册登录后https://openai.com/，点击头像 - > View API Keys - >Create new secret key

VScode 快捷键ctrl+shift+p，搜索“chatGPT：切换…….”,点击后复制粘贴OpenAI获取到的Key，然后回车就可以切换到国外模式。

总结：

可玩性最高的是网页在线模拟的GPT3.5。不是传统的聊天机器人，非常成熟，非常强大。语言组织能力、文本水平、逻辑能力，可以说已经令我惊艳。但是需要代理。

GPT3.0的API调用差强人意，胜在后续使用无需代理。

目前国内已经存在大量“平替“、”山寨”，使用的AI对话模型质量不及ChatGPT，还需要每回答一个问题看一则广告，甚至充钱付费。还有卖号，买教程的。本教程完全免费无保留希望能够帮到大家，有问题留言带邮，我会尽力解答。
